# Network Basics 0

This directory contains basic network concepts and scripts.

## Files

- **0-OSI_model**: Answer to the OSI model question.
- **1-types_of_network**: Types of networks.
- **2-MAC_and_IP_address**: MAC and IP address differences.
- **3-UDP_and_TCP**: UDP and TCP protocols.
- **4-TCP_and_UDP_ports**: Bash script to display listening TCP and UDP ports with PID and program name.
- **5-is_the_host_on_the_network**: Bash script to ping a host 5 times.
