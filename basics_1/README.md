# Network Basics 1

This directory contains scripts for basic network configuration and information retrieval.

## Files

- **0-change_your_home_IP**: Bash script that configures the hosts file to redirect localhost to 127.0.0.2 and facebook.com to 8.8.8.8.
- **1-show_attached_IPs**: Bash script that displays all active IPv4 IPs on the machine.
- **2-port_listening_on_localhost**: Bash script that listens on port 98 on localhost using netcat.
